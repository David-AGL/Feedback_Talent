# Feedback_Talent

## 📝 Descripción
Este proyecto consiste en una **plataforma web** donde **candidatos y empleados** pueden dar feedback sobre procesos de selección y experiencias internas en las empresas.  
Al mismo tiempo, las **marcas empleadoras** pueden acceder a estas opiniones mediante **gráficas, filtros y paneles de análisis**, con el fin de mejorar sus procesos de reclutamiento y clima organizacional.  

## 🎯 Objetivos
- Dar voz a candidatos y empleados en procesos de selección e internos.  
- Permitir a empresas recopilar información útil para la mejora continua.  
- Garantizar una experiencia de usuario intuitiva, rápida y segura.  
- Asegurar el cumplimiento de normativas de protección de datos.  

## 🚀 Funcionalidades Principales
- ✍️ **Feedback de candidatos y empleados** sobre procesos de selección e internos.  
- ✅ Confirmación de guardado y edición de opiniones.  
- 🔎 **Consulta de opiniones y calificaciones** por empresa, con filtros por fecha y categoría.  
- 📊 **Panel para marcas empleadoras** con gráficas de promedios, medianas y distribuciones.  
- 🤖 **Chatbot interactivo** para ingresar feedback de manera guiada.  
- 🗑️ Gestión de feedback (ver, editar, eliminar).  
- 👤 **Perfiles de usuario** (candidatos, empleados, empresas, administradores).  
- 🔐 **Seguridad y roles** con control de accesos.  
- 🌐 **Optimización SEO** (metaetiquetas, URLs amigables, sitemap).  
- 📂 Exportación de feedbacks en PDF y CSV.  

## 🏗️ Tecnologías
- **Frontend:** React  
- **Backend:** Node.js + Express  
- **Base de Datos:** MongoDB
